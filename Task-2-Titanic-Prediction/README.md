# Titanic Survival Prediction

## Objective
Predict whether a passenger survived using machine learning.

## Dataset
Titanic dataset with features like age, gender, class, fare, etc.

## Feature Engineering
- Extracted Title from Name
- Created FamilySize
- Created HasCabin feature

## Data Preprocessing
- Filled missing values (Age, Fare, Embarked)
- Encoded categorical variables

## Models Used
- Logistic Regression
- Random Forest

## Accuracy
~80%

## Conclusion
Model predicts survival based on key features like Sex, Fare, and Pclass.
